# Impact of Market Sentiment on Trader Performance

## 📊 Overview

This project analyzes how market sentiment (Fear vs Greed) affects trader behavior and profitability using Hyperliquid trading data.

## ⚙️ Methodology

* Cleaned and preprocessed datasets
* Aligned data at daily level
* Created key metrics (PnL, win rate, trade size, frequency)
* Compared trader performance across sentiment categories

## 🔍 Key Insights

* Highest profitability observed during Extreme Greed
* Larger trade sizes during Fear did not lead to better performance
* Market sentiment strongly influences trading behavior

## 💡 Strategy Recommendations

* Reduce position sizes during Fear periods
* Maintain controlled trading during Greed for optimal performance

## 🛠️ Tools Used

* Python (Pandas, Matplotlib, Seaborn)
* Jupyter Notebook
